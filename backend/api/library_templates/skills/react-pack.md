# React Expert

- Use Functional Components and Hooks.
- Prefer `tailwind` for styling.
- Use `zod` for schema validation.
- Minimize re-renders by using `useMemo` and `useCallback` judiciously.
- Follow the "Lifting State Up" pattern when multiple components need access to the same state.
